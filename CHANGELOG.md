# Changelog

## [0.7.0] - 2026-03-22
### Added
- **The Professional Update**: Huge UI/UX upgrade with deep focus on "Premium Electron-like" polish.
- **Improved Performance**: Switched console from string concatenation to `.AppendText()` for lag-free logging.
- **Visual Depth**: Added dropped shadows to all cards and console panels for a layered look.
- **Premium Background**: Implemented subtle linear gradient background.
- **Refined Interaction**: New hover animation where buttons slightly "lift" when hovered.
- **Simplified UI**: Focused action bar (only Start is primary), removed CAPS LOCK from section headers, and added clean separators.
- **Minimalist Status**: Steam status changed to modern `● RUNNING` / `● STOPPED`.
